# Tic Tac Toe Game

Play the classic game on your browser! Get 3 in a row to win.

## 🎮 How to Play

1. Open `index.html` in your web browser
2. Click on an empty box (you are O)
3. Computer plays as X
4. Get 3 in a row to win
5. Click Reset to play again

## 🎯 Game Rules

Win by getting 3 marks in a row:
- Horizontal (left to right)
- Vertical (up to down) 
- Diagonal (corner to corner)

The game board:
```
0 | 1 | 2
---------
3 | 4 | 5
---------
6 | 7 | 8
```
